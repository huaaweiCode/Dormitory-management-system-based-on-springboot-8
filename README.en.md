# 基于springboot的宿舍管理系统8

#### Description
基于springboot的宿舍管理系统8拥有三种角色

- 管理员：宿舍楼管理、学生信息管理、职工信息管理等
- 宿管：个人信息修改、来访人员管理、离校登记管理、入校登记管理、历史报修单、历史来访记录、历史宿舍评分、宿舍评分录入、查看未处理及处理中及已完成报修单、学生信息查询等功能
- 学生：查看宿舍评分、个人信息修改、返校登记、离校登记、申请报修、历史报修等
